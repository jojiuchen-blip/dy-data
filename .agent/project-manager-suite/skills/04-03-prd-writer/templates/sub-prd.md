  X.1 用户体验        — 数据来源声明 + ASCII 布局 + 交互说明 + 前端职责声明

  UX → 服务端逻辑 → 数据链路 → 异常 → 配置 → 验收
  = 用户看到什么 → 服务端怎么算 → 数据从哪来 → 出错怎么办 → 运营怎么改 → 怎么验

  - X.5 配置项：该子区域无配置页时省略

  模块化原则：
  配置项跟着功能子区域走，不集中放末尾。
