# 经营数据智能看板项目简介

## 一、项目概述

本项目建设了一套经营数据智能看板，用于展示抖音服务产品的销售、核销、退款和后续分账情况。看板以 HTML 网页形式发布到腾讯云，手机和电脑都可以直接打开查看。

当前已完成“抖音服务产品销售数据看板”，后续将在同一个 HTML 页面中加入“分账看板”，形成一个统一的经营数据入口。

## 二、看板一：抖音服务产品销售数据

该看板主要基于抖音来客订单、券状态、商品类型、金额和时间字段生成。

核心逻辑：

- 按订单 ID 去重，避免重复统计。
- 剔除支付取消订单，统计有效下单量。
- 根据券状态判断已核销、待核销和已退款订单。
- 汇总核销金额、核销量、退货量、核销率、退货率。
- 根据下单时间和更新时间计算核销周期。
- 支持按日期和商品类型筛选。

主要作用：

- 快速查看服务产品整体经营情况。
- 判断哪些产品卖得好、核销快、退款高。
- 观察每月下单和核销趋势。
- 辅助活动复盘、门店运营和产品优化。

## 三、看板二：分账数据

后续分账看板将与销售看板放在同一个 HTML 页面中，形成“销售结果 + 财务分账”的完整链路。

规划逻辑：

- 将订单、核销、退款和分账规则进行关联。
- 计算应分账金额、已分账金额和未分账金额。
- 按月份、门店、商品类型或分账对象汇总。
- 识别退款冲减、漏分账、分账差异等异常。

主要作用：

- 快速查看每月应分账和已分账金额。
- 减少财务人工核对 Excel 的工作量。
- 帮助管理层同时掌握经营结果和资金分配结果。

## 四、统一 HTML 看板的价值

两个看板放在一个 HTML 页面中，可以实现：

- 一个链接查看销售、核销、退款和分账数据。
- 手机端直接访问，无需安装软件。
- 数据更新后重新上传即可刷新。
- 后续可继续扩展门店排名、异常预警、利润分析等模块。

## 五、AI 在项目中的应用

本项目通过 AI 辅助完成了从业务需求到看板落地的开发过程：

- AI 协助梳理业务口径，例如有效订单、已核销、已退款、核销率等定义。
- AI 协助编写数据处理脚本，实现订单去重、字段清洗、指标汇总。
- AI 协助生成 HTML 看板页面，包括 KPI、趋势图、商品表和核销周期分布。
- AI 协助优化手机端访问问题，将大文件改为聚合数据展示，提高加载稳定性。
- AI 协助接入腾讯云 COS，实现看板网页发布和一键上传更新。
- AI 协助排查核销周期为空、脚本编码、云端上传等问题。

## 六、使用方式

日常查看：

- 手机或电脑打开腾讯云发布的网址。
- 页面支持日期筛选、商品类型筛选和刷新数据。

数据更新：

- 电脑运行一键脚本，自动重新生成看板并上传腾讯云。
- 手机上点击“刷新数据”即可查看最新版本。

## 七、项目成果

该项目将原本依赖人工导表和 Excel 统计的流程，升级为自动化、可视化、可发布的经营数据工具。它既能服务日常经营管理，也能体现 AI 在数据分析、工具开发和业务数字化中的实际应用能力。
