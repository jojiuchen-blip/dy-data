03

开始前请准备以下信息

提前确认门店资料，可以减少激活过程中的中断。

  • 账户所属 ID 对应导出字段“账户所属id”
  • 所属账户关联 POI ID 对应导出字段“所属账户关联poi_id”，即门店 POI ID
  • 用于后续登录的账号名和密码 账号名和密码由您自行设置，激活成功后用于后续登录。

两个 ID 必须来自导出文件中的同一条子机构经营号记录，仅用于账号激活时核验门店身份。

04

如何完成账号激活

先核验门店身份，再设置登录账号和密码。

  1. 1

    进入测试环境，打开账号登录页面并切换到“账号激活”。

  2. 2

    在第一屏填写账户所属 ID 和所属账户关联 POI ID。两个字段右侧的问号可查看获取方式。

  3. 3

    点击“激活状态核验”。如两个 ID 不匹配，页面会显示“账户所属ID和所属账户关联POI ID不正确”；如账号已有密码，页面会显示“账户已激活过，需要前往账户登录”。

  4. 4

    核验通过且账户尚未激活时进入第二屏，设置仅包含数字和英文字母的账号名，并输入密码和确认密码。

  5. 5

    确认信息无误后，点击“设置账号并完成激活”。

激活成功后，系统会自动关联门店并建立数据权限。后续使用您设置的账号名和密码登录系统。

05

如何获取账户所属 ID 和所属账户关联 POI ID

两项信息均需从抖音来客“子机构经营号”的导出数据中获取。

  1. 1

    进入子机构经营号页面

    店端登录抖音来客，依次进入【店铺管理】-【抖音号管理】-【子机构经营号】。

    直接打开子机构经营号页面
  2. 2

    确认门店已有子机构经营号

    在列表中查找当前门店对应的子机构经营号。若不存在，点击【创建新账号】，按页面提示完成创建。

  3. 3

    导出并查找两个 ID

    点击页面右上角的【导出数据】。打开导出的文件，找到当前门店对应的子机构门店号记录，并从同一行获取“账户所属id”和“所属账户关联poi_id”。

请确保两个 ID 来自导出文件中的同一条子机构经营号记录。

06

激活后可以做什么

账号激活后，系统会为门店建立对应的数据权限。

  • 激活成功后，系统会根据账号对应的门店建立数据权限。登录后，您只能查看当前账号有权访问的门店数据。
  • 账户所属 ID 和所属账户关联 POI ID 仅用于激活核验；后续使用自行设置的账号名和密码登录。
  • 忘记密码时，先输入账户所属 ID 和所属账户关联 POI ID。账户信息核验通过后，再设置新密码；原账号名保持不变。

07

测试环境账号会保留吗？

正式环境迁移说明

会。您现在在测试环境完成的账号激活信息，会在系统迁移至内网正式环境时一并保留。

正常情况下，正式环境启用后无需再次激活账号。正式内网环境的启用时间和访问安排将另行通知。

08

常见问题

查看账号激活过程中常见问题的处理方式。

已经激活过账号怎么办？

状态核验会显示“账户已激活过，需要前往账户登录”。直接前往账号登录；如忘记密码，可使用同一组双 ID 核验后设置新密码。

账号激活失败怎么办？

如果显示“账户所属ID和所属账户关联POI ID不正确”，请确认两个 ID 来自导出文件中的同一条认证成功记录。忘记密码时如显示“账户尚未激活，请先完成账号激活”，请先返回账号激活入口。
