# 抖音来客券核销清单自动导出

## 1. 先设置环境变量

在 PowerShell 里执行一次，替换成你自己的抖音开放平台配置：

```powershell
[Environment]::SetEnvironmentVariable("DOUYIN_APP_ID", "你的 client_key", "User")
[Environment]::SetEnvironmentVariable("DOUYIN_APP_SECRET", "你的 client_secret", "User")
[Environment]::SetEnvironmentVariable("DOUYIN_ACCOUNT_ID", "你的抖音来客商家账号ID", "User")
```

如果只导出指定门店，可额外设置，多个门店用英文逗号分隔：

```powershell
[Environment]::SetEnvironmentVariable("DOUYIN_POI_IDS", "门店ID1,门店ID2", "User")
```

设置后重新打开 PowerShell，让新环境变量生效。

## 2. 手动测试

```powershell
python .\douyin_verify_record_export.py
```

默认导出上一自然周：上周一 00:00 到本周一 00:00，不包含本周一 00:00 之后的数据。

导出目录可通过 `config.local.json` 的 `paths.verify_save_dir` 或环境变量 `DOUYIN_VERIFY_SAVE_DIR` 配置。

```text
exports\verify
```

## 3. 注册每周一自动导出

默认每周一 09:00 执行：

```powershell
powershell -ExecutionPolicy Bypass -File ".\setup_weekly_verify_export_task.ps1"
```

如果要改成每周一 08:30：

```powershell
powershell -ExecutionPolicy Bypass -File ".\setup_weekly_verify_export_task.ps1" -RunAt "08:30"
```

## 4. 临时导出指定时间段

开始和结束时间是左闭右开区间，例如下面导出 2026-05-25 00:00:00 到 2026-06-01 00:00:00：

```powershell
$env:DOUYIN_VERIFY_START = "2026-05-25 00:00:00"
$env:DOUYIN_VERIFY_END = "2026-06-01 00:00:00"
python .\douyin_verify_record_export.py
```
