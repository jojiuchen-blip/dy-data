销售订单数量
订单销售时间落入所选月份、销售归属门店明确，且排除退款、关闭、撤销券明细后仍至少保留一张有效券，即计为有效销售订单；包含尚未核销的订单，每个 order_id 只计一次。
销售订单金额
与销售订单数量使用同一批有效 order_id，每个订单只取一次订单表整单实收金额。一单多券时不会重复金额；部分券退款但订单仍有有效券时，仍按该订单表记录的整单实收金额统计。
核销收入
券核销时间落入所选月份且实际核销门店明确时，按实际核销门店汇总有效券的实收金额；每张券只汇总一次，退款、关闭或撤销券不计入。
厂端激励推广费
按销售归属门店归集本月完成的有效核销券。每张券以券级实收金额乘所属商品类型的推广费比例，四舍五入到分后再汇总；推广费比例为空的商品类型不进入该指标。
排名规则
按销售订单数量从高到低排名；数量相同时按门店 ID 固定顺序排列，保证同一批数据重复查询时排名稳定。销售指标按销售时间，核销收入和推广费按核销时间。
