# 抖音来客退款单导出

## 1. 先设置环境变量

在 PowerShell 里执行一次，替换成你的抖音开放平台配置：

```powershell
[Environment]::SetEnvironmentVariable("DOUYIN_APP_ID", "你的 client_key", "User")
[Environment]::SetEnvironmentVariable("DOUYIN_APP_SECRET", "你的 client_secret", "User")
[Environment]::SetEnvironmentVariable("DOUYIN_ACCOUNT_ID", "你的抖音来客商户账号ID", "User")
```

设置后重新打开 PowerShell，让新环境变量生效。

## 2. 手动导出上一自然周

```powershell
python .\douyin_refund_export.py
```

默认导出上一自然周：上周一 00:00 到本周一 00:00，不包含本周一 00:00 之后的数据。

导出目录可通过 `config.local.json` 的 `paths.refund_save_dir` 或环境变量 `DOUYIN_REFUND_SAVE_DIR` 配置。

```text
exports\refund
```

## 3. 导出指定时间段

开始和结束时间是左闭右开区间。例如导出 2026-05-25 00:00:00 到 2026-06-01 00:00:00：

```powershell
$env:DOUYIN_REFUND_START = "2026-05-25 00:00:00"
$env:DOUYIN_REFUND_END = "2026-06-01 00:00:00"
python .\douyin_refund_export.py
```

## 4. 可选参数

按退款完成时间查询，而不是按退款创建时间查询：

```powershell
$env:DOUYIN_REFUND_TIME_FIELD = "complete"
```

只导出指定退款状态，多个状态用英文逗号分隔：

```powershell
$env:DOUYIN_REFUND_STATUSES = "50"
```

状态码说明以抖音开放平台文档为准，脚本内已内置常见状态中文映射。

自定义保存目录：

```powershell
$env:DOUYIN_REFUND_SAVE_DIR = "exports\refund"
```

只测试前几页，不跑完整导出：

```powershell
$env:DOUYIN_REFUND_MAX_PAGES = "1"
```

只测试订单映射前几页：

```powershell
$env:DOUYIN_REFUND_MAX_ORDER_PAGES = "1"
```

脚本默认会按以下 SKU_ID 筛选退款单，并新增 `商品类型` 列：268保养、168保养、漆面、蒸发箱清洗、外循环清洗。若要临时关闭 SKU 筛选：

```powershell
$env:DOUYIN_REFUND_FILTER_TARGET_SKUS = "0"
```
